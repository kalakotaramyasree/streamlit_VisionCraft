# 🎨 VisionCraft – Image Editing App

A simple and interactive image editing web app built using Streamlit and OpenCV.  
Users can upload images and apply different image processing techniques easily.

---

## 🚀 Features
- Upload an image
- Apply Blur
- Adjust Brightness
- Sharpen Image
- Convert to Grayscale
- Real-time preview

---

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenCV (cv2)
- NumPy

---

## 📂 Project Structure
streamlit_visioncraft/
│── app.py
│── requirements.txt
│── README.md

---

## ▶️ How to Run Locally

1. Clone the repository & Install,Run dependecies:

git clone https://github.com/your-username/streamlit_visioncraft.git
cd streamlit_visioncraft
pip install -r requirements.txt
Run the app:
streamlit run app.py
